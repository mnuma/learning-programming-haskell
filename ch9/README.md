# 対話プログラム

## 9.1 対話

- 純粋関数と副作用
- 対話プログラムは基本的に副作用を起こす

## 9.2 入出力の型

- Haskellにおいて対話プログラムとは「ある状態の世界」を取り「別な状態の世界」を返す純粋な関数
  - type IO = World -> World
- 一般化すると type IO a = World -> (a, World)
- 型 IO a を持つ式を「アクション」と呼ぶ
- IO Char は文字を返すアクション
- IO () は結果に意味がないことを示すアクション
- Haskellでは副作用を起こす処理をIOという型に包む
  - 参照透過性ってなんや？
  - 同じ引数に対しては常に同じ値を返すということ
